# Aplicação de Tarefas

Esta é uma aplicação React para gerenciar tarefas (to-do list), utilizando rotas, estado, efeito e contexto.

## Funcionalidades

- Adicionar tarefas
- Visualizar lista de tarefas
- Editar tarefas
- Excluir tarefas
- Persistência com localStorage

## Como executar

1. Instale as dependências: `npm install`
2. Execute o servidor de desenvolvimento: `npm run dev`
3. Abra o navegador em http://localhost:5173

## Tecnologias

- React
- React Router
- Context API
- Vite
